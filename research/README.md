# Research Source Files

Put production-facing research files here.

Recommended pattern:

- Keep a human-readable export:
  - `stock-analysis.html`
  - `ai-supply-chain.html`
- Keep a machine-friendly summary:
  - `stock-analysis.json`
  - `ai-supply-chain.json`

The daily brief should read the machine-friendly files first and use the HTML exports as supporting context.
